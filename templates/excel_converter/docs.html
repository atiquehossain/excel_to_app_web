Getting Started

The Excel to Dart Code Generator is a web-based tool that helps you convert Excel spreadsheet data into Dart/Flutter code. Follow these steps to get started:

  1. Prepare your Excel file with the required data structure
  2. Upload your Excel file using the web interface
  3. Enter a class name for the generated code
  4. Click "Generate Code" or "Preview" to see the results
  5. Download the generated Dart code

Excel File Format

Your Excel file should follow this structure:

  • First row should contain column headers
  • Each column represents a field in your Dart class
  • Supported field types:
    • Dropdown
    • Multiple choice
    • Radio
    • Text
    • Number
    • Image

Features

  • Convert Excel data to Dart/Flutter code
  • Support for multiple field types
  • Multi-language support (English, Tamil, Sinhala)
  • Preview generated code before downloading
  • Syntax highlighting for generated code
  • Responsive web interface

Configuration

You can customize the behavior by creating a `config.yaml` file: 

excel:
  default_sheet: "Sheet1"
  skip_hidden_rows: true

dart:
  output_folder: "generated_code"
  template_folder: "templates"
  null_safety: true

languages:
  - English
  - Tamil
  - Sinhala

Frequently Asked Questions

What Excel formats are supported?

We support .xlsx, .xls, and .csv files.

Can I customize the generated code?

Yes, you can customize the templates in the templates directory.

Is there a size limit for Excel files?

The maximum file size is 16MB.
